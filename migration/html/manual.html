1. Login

Jika anda sudah berhasil login, maka akan muncul halaman beranda yang akan menampilkan jadwal presensi

2. Masuk ke menu presensi

Pastikan sudah memberikan izin akses kamera, GPS dan Berkas

3. Tekan tombol kamera untuk melakukan presensi

Tambahkan berkas kinerja jika diwajibkan mengunggah

4. Tekan tombol simpan untuk mengirim data presensi

5. Masuk ke menu riwayat

Riwayat presensi anda akan tampil disini
