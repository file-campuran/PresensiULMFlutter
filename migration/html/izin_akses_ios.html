1. Mengaktifkan Layanan Lokasi

Buka menu/aplikasi pengaturan. Aplikasi pengaturan dapat ditemukan di salah satu home screen perangkat. Ikonnya tampak seperti beberapa buah roda gigi. Jika tidak ditampilkan di home screen, aplikasi pengaturan mungkin berada dalam direktori berlabel “Utilities”.
  • Jika Anda tidak bisa menemukan aplikasi pengaturan, geser layar ke arah bawah ketika Anda berada di home screen untuk membuka fitur Spotlight Search, kemudian carilah “Settings” pada bilah pencarian yang ditampilkan.

2. Privacy

Pilih “Privacy”. Pilihan tersebut berada di bagian bawah daftar pengaturan, di kelompok pengaturan ketiga.

3. Location Services

Sentuh “Location Services”. Setelah itu, menu pengaturan layanan akan ditampilkan.

4. Allow Location Access

Sentuh salah satu aplikasi dari daftar yang ditampilkan untuk mengaktifkan layanan lokasi untuk aplikasi tersebut. Ketika Anda memilih aplikasi dari daftar, Anda bisa melihat pilihan layanan lokasi yang tersedia untuk aplikasi tersebut.
  • Pilih "Never" untuk menonaktifkan layanan lokasi secara penuh untuk aplikasi tersebut.
  • Pilih "While Using" untuk membatasi layanan lokasi agar hanya diaktifkan ketika aplikasi sedang dibuka atau digunakan saja.
  • Pilih "Always" untuk mengizinkan aplikasi menggunakan layanan lokasi setiap saat. Pilihan ini hanya tersedia untuk aplikasi-aplikasi yang dapat berjalan di belakang (background), seperti aplikasi Weather.
